# ec-clean 🧹

**電商訂單資料清理工具** — 針對官網後台匯出的訂單 Excel 進行自動化整理。

## 功能流程

三步驟一鍵完成：

| 步驟 | 說明 |
|------|------|
| **Step 1** | 刪除完全空白列 |
| **Step 2** | 子項目自動補上父訂單資料（含儲存格格式） |
| **Step 3** | 商品編號智慧校正（非數字編號 → 取自商品型號/ISBN） |

## 安裝需求

```bash
pip install openpyxl
```

## 使用方式

```bash
python ec_clean.py <來源訂單.xlsx> [輸出檔名.xlsx]
```

### 參數

| 參數 | 說明 |
|------|------|
| `來源訂單.xlsx` | 官網後台匯出的訂單 Excel（必填，60 欄格式） |
| `輸出檔名.xlsx` | 自訂輸出檔名（選填，預設為 `<原始檔名>_已整理.xlsx`） |

### 範例

```bash
python ec_clean.py 訂單.xlsx
python ec_clean.py 訂單.xlsx 已整理_訂單.xlsx
```

## 輸出範例

```
原始資料：9 列 x 60 欄

[Step 1] 刪除空白列...
  -> 已刪除 3 列空白

[Step 2] 子項目補父訂單資料...
  -> Row 6: 子項「YLE A2 Flyers…」補入 31 欄（含格式）
  -> 共 1 列子項目補齊

[Step 3] 商品編號智慧校正...
  -> Row 3: 「Qsfs_01」→「9780194903943」（取自商品型號）
  -> Row 5: 「YLE_A1」→「9781316617533」（取自商品型號）
  -> Row 6: 「YLE_A2」→「9781316617588」（取自商品型號）
  -> 共 3 列商品編號校正

==================================================
整理完成！
  輸出: 訂單_已整理.xlsx
  最終: 6 列 x 60 欄（5 筆資料）
```

## 注意事項

1. **原始檔案會自動備份**（`<原始檔名>_backup.xlsx`）
2. 僅支援官網後台匯出的標準 60 欄格式
3. 標題列（第 1 列）保持不變，資料從第 2 列開始處理
4. 商品編號校正僅在商品型號為純數字時執行，避免誤寫

# 電商訂單資料清理 (ec-clean)

## 腳本位置
`~/.config/opencode/skills/ec-clean/ec_clean.py`

## 使用方式
```powershell
python ~/.config/opencode/skills/ec-clean/ec_clean.py <來源訂單.xlsx> [輸出檔名.xlsx]
```

### 參數
| 參數 | 說明 |
|------|------|
| `來源訂單.xlsx` | 官網後台匯出的訂單 Excel（必填，60 欄格式） |
| `輸出檔名.xlsx` | 自訂輸出檔名（選填，預設為 `<原始檔名>_已整理.xlsx`） |

### 範例
```powershell
python ~/.config/opencode/skills/ec-clean/ec_clean.py 訂單.xlsx
python ~/.config/opencode/skills/ec-clean/ec_clean.py 訂單.xlsx 已整理_訂單.xlsx
```

## 依賴套件
- `openpyxl`（處理 Excel）

若無 openpyxl，請先安裝：
```powershell
pip install openpyxl
```

---

## 處理流程

### Step 1：刪除完全空白列
從最後一列往上掃描，若該列所有欄位均為 `None`，則判定為空白列並刪除。

### Step 2：子項目補父訂單資料（含格式）
1. 逐列掃描，當第 1 欄（訂單編號）有值 → 記錄為當前父訂單（60 欄的**值** + **number_format**）
2. 當第 1 欄無值但第 31 欄（商品名稱）有值 → 判定為子項目
3. 子項目空白欄位用父訂單補上，**連同儲存格格式一併複製**，確保日期、數字顯示一致

### Step 3：商品編號智慧校正
對 AF（第 32 欄 = 商品編號）做兩層判斷：
1. 若 AF **不是純數字**（如 `Qsfs_01`、`YLE_A1` 等文字代碼）
2. 且 AI（第 35 欄 = 商品型號）**是純數字**（如 ISBN）
3. 則將 AI 的值與格式複製到 AF
4. 若 AI 也不是純數字 → 不動作

## 注意事項
1. **原始檔案會自動備份**（`<原始檔名>_backup.xlsx`）
2. 僅支援官網後台匯出的標準 60 欄格式
3. 標題列（第 1 列）保持不變，資料從第 2 列開始處理
4. 商品編號校正僅在 AI 為純數字時執行，避免誤寫
